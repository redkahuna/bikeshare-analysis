US Bikeshare Data Analysis
========================

This Python script explores bike sharing data from three cities in the US: Chicago, New York City, and Washington. It reads data from .csv files and allows users to filter the data by month and day of the week to gain insights into the most common times, stations, and trip routes.

Required Libraries
------------------

The script requires the following libraries:

- Pandas
- Numpy
- Time

Files
-----

The script uses the following data files:

- chicago.csv
- new_york_city.csv
- washington.csv
- bikeshare.py

How to Run the Script
---------------------

To run the script, open a command line window in the project directory and type the following command:

python bikeshare.py
